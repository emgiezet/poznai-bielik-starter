# multi-step-starter (Python)

Dwa wywołania Bielika w łańcuchu: najpierw ekstrakcja struktury, potem transformacja.

**Jak uruchomić:**
```bash
cd python/multi-step-starter
python main.py przyklad-notatki.txt
```

**3 pomysły jak to rozszerzyć:**
1. **Second brain** — wrzuć bazgroły z wykładu, dostaniesz tagi + sugestię struktury folderów (przykład jest gotowy w `przyklad-notatki.txt`)
2. Zmień etap 2 na `na podstawie tej struktury napisz poetycki post na social media` — dane → treść
3. Zmień oba prompty na fiszki: etap 1 wyciąga pojęcia, etap 2 tworzy pytanie + odpowiedź dla każdego
