# chat-starter (Python)

Rozmowa z Bielikiem w pętli — zmieniasz system prompt i masz własnego asystenta z charakterem.

**Jak uruchomić:**
```bash
cd python/chat-starter
python main.py
```

Wpisz `q` albo wciśnij Ctrl+C żeby wyjść.

**3 pomysły jak to rozszerzyć:**
1. Zmień `SYSTEM_PROMPT` żeby Bielik był brutalnie szczerym kumplem oceniającym rozmowy z Tindera
2. Dodaj `wciel się w polskiego wujka z Piły` — gotowy Janusz mode
3. Zmień na `wciel się w moją mamę, kontrolującą i pasywno-agresywną` — symulator trudnej rozmowy
